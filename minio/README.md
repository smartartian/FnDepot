# MINIO（20221117）

MinIO 是一款高性能的对象存储服务器，兼容 Amazon S3 API。此为 2022-11-17 定制版，专为飞牛OS优化。

## 应用信息

- 应用名：MINIO（20221117）
- 包名：`minio`
- 当前版本：`1.0.0`
- 发布者：ShawnLee
- 原维护站点：<https://github.com/smartartian>

## 简介

MinIO 提供高性能、高可用的对象存储服务，完全兼容 Amazon S3 API。适用于数据备份、静态资源托管、大数据分析等场景。

默认访问密钥：`minioadmin/minioadmin`
默认控制台端口：`9001`
默认API端口：`9000`

## 安装说明

从 FnDepot 安装后即可使用。当前仓库内提供 `minio.fpk` 安装包。
