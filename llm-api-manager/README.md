# LLM API Manager

LLM API 管理器，提供统一的 API 接口管理、密钥管理和调用统计功能。

## 应用信息

- 应用名：LLM API Manager
- 包名：`llm-api-manager`
- 当前版本：`1.0.0`
- 发布者：ShawnLee

## 简介

LLM API Manager 是一款高效的 LLM API 管理工具，支持多种大语言模型提供商，提供统一的 API 接口、密钥管理和调用统计功能。

默认端口：`8000`

## Docker 配置

- 镜像：`llm-api-manager`
- 端口映射：`8000:8000`
- 数据卷：`/data:/app/data`

## 安装说明

从 FnDepot 安装后即可使用。
